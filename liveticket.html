About LiveTicket™

LiveTicket™以及LiveTicket™ Auth System是我们专有的门票认证及检票技术
LiveTicket™ and the LiveTicket™ Auth System is our elclusive ticket authentication and check-in technology.

LiveTicket™采用了Liq™ Backend以及DataMatrix，用提供高效，可靠，且尊重隐私的检票体验；每一张LiveTicket™都有着独一无二的DataMatrix编码，有效的编码组合将会被记录在Liq™ Backend中
LiveTicket™ uses Liq™ Backend and DataMatrix to provide a high-efficiency, reliable, and privacy respecting ticket check-in experience; each LiveTicket™ owns a unique DataMatrix code which vaild code combination will be recorded to Liq™ Backend.
若复制我们已发行的LiveTicket™，仅有第一张具有同一个DataMatrix码的LiveTicket™可以通过认证；任何未在Liq™ Backend登记的DataMatrix码将会被系统拒绝；在此机制下试图伪造LiveTicket™相当困难
In case a LiveTicket™ we issued gets dumplicated, only the first LiveTicket™ with the same DataMatrix code is able to pass the verification; any DataMatrix code that was not registered in Liq™ Backend will be rejected by our system; successful counterfeit is hard to achieve in this mechanism.

LiveTicket™技术同样可以辅助活动组织者更好的进行统计和分析；例如通过不同方式分发的LiveTicket™的入场率；
LiveTicket™ technology could assist event organisers with better statistics and analysis; for example the attendance rate of LiveTickets™ distributed by different means;
此技术同样方便了活动组织者进行管理；每一张LiveTicket™都可以在通过检票前修改类型，或撤回有效标记；
This technology also makes it easier for event organisers to manage; each LiveTicket™ can be changed in type or revoke from validation before passing through the check-in device;
LiveTicket™认证系统同样记录了每张LiveTicket™的设计样式，为未来发行与用户持有的LiveTicket™设计一致的NFT提供了可能
LiveTicket™ authentication system also records the design of each LiveTicket™, making it possible to issue future NFTs that match the design of the LiveTicket™ held by the user.

相对于其他的需要提供用户证件的检票系统，例如eplus，或是需要用户提供面部识别数据的检票系统，例如AUTH thru，LiveTicket™更好的做到了尊重用户隐私；
LiveTicket™ respects user privacy better than other ticketing systems that require user identification, such as eplus, or that require user facial recognition data, such as AUTH thru.
在LiveTicket™系统中被写入的，与营销或统计分析无关的数据，例如邮寄地址，将会在写入数据库一段时间后被清除，详情请检阅我们的条款和隐私声明
Data written in the LiveTicket™ system that is not relevant to marketing or statistical analysis, such as mailing addresses, will be erased from our database after a period of time; details can be found at our Terms & Privacy Notice.

若要访问LiveTicket™验证系统，请前往LiveTicket™验证系统
To access LiveTicket™ Auth System, please proceed to LiveTicket™ Auth System.
